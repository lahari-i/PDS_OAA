# README – Data Wrangling (Principles of Data Science)

## Overview
This project focuses on preparing the **Crop_Production** dataset for analysis by cleaning, organizing, and validating the data.  
It was done as part of a Data Science workshop to understand dataset validation and transformation using Python tools.

---

## Steps Involved
1. **Discovering:** Explored the dataset and identified errors or missing values.  
2. **Structuring:** Renamed columns and organized data for analysis.  
3. **Cleaning:** Fixed formatting issues and handled missing or incorrect data.  
4. **Enrichment:** Added new useful columns and merged external data.  
5. **Validation:** Checked accuracy and consistency of all values.

---

- `Crop_Production.csv`  
- `Crop_Production1.csv`  
- `city.csv`
